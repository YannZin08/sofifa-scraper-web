# Guia de Deploy Permanente - SoFIFA Web Scraper\n\nPara transformar este projeto em um site permanente e acessível de qualquer lugar, siga os passos abaixo. Recomendamos o uso da **Vercel** ou **Railway**.\n\n## Opção 1: Deploy na Vercel (Recomendado para o Frontend)\n\n1. Acesse [Vercel.com](https://vercel.com) e faça login com seu GitHub.\n2. Clique em **'Add New'** -> **'Project'**.\n3. Importe o repositório 'sofifa-scraper-web'.\n4. Nas configurações do projeto:\n   * **Framework Preset:** Vite\n   * **Build Command:** 'pnpm build'\n   * **Output Directory:** 'dist/client'\n5. Clique em **'Deploy'**.\n\n## Opção 2: Deploy no Railway (Recomendado para o Backend com Puppeteer)\n\nComo o scraper usa o **Puppeteer** (navegador automatizado), ele exige um servidor que suporte a instalação do Chromium. O Railway é excelente para isso:\n\n1. Acesse [Railway.app](https://railway.app) e conecte seu GitHub.\n2. Clique em **'New Project'** -> **'Deploy from GitHub repo'**.\n3. Selecione 'sofifa-scraper-web'.\n4. O Railway detectará automaticamente o 'package.json' e iniciará o deploy.\n5. O site ficará online em uma URL como 'seu-projeto.up.railway.app'.\n\n---\n*Este guia foi gerado automaticamente pela Manus AI para o projeto SoFIFA Web Scraper.*
